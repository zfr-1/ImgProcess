# 数字图像处理作业：图像检索系统

一开始的计划为利用直方图进行图像检索。

~~缺点：如果数据集内的图像十分相似，这会导致直方图也十分相似，而出现按相似度排序失败的情况。~~

所以追加利用纹理特征通过 LBP 方法来进行提取，成功率大大上升。

## How to run

Build success on Matlab R2015b & R2019b.

Run `window.m`.

Plase switch your MATLAB locale to GBK.
